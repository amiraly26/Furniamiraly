# Furni EcomV2

Furniture and home decor ecommerce website built with:

- Next.js frontend in `ecomv2-frontend`
- Payload CMS backend in `payload-backend`
- Neon PostgreSQL database

## Local URLs

- Frontend: `http://localhost:3000`
- Payload admin: `http://localhost:1337/admin`

## Run Locally

Backend:

```bash
cd payload-backend
npm run dev
```

Frontend:

```bash
cd ecomv2-frontend
npm run dev
```

## Admin

Create or use the Payload admin user from the backend setup, then manage products, categories, orders, and reviews from the Payload dashboard.
